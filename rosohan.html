Озеро Росохан

Найбільше озеро Ґорґан - лайтовий маршрут для новачків

1600 ₴ Виїзд з Чернівців

Програма подорожі

06:00

Виїзд з Чернівців

Збираємося та вирушаємо до найбільшого озера Ґорґан. Це лайтовий маршрут, який здолає кожен новачок 😉

Ранок

Зупинки по дорозі

Будуть зупинки по дорозі на заправках для відпочинку та поповнення запасів.

Полудень

Дорога через горганський ліс

Підїжаємо максимально близько до озера за можливості погодних умов. Дорога йде через красивий горганський ліс.

13:00

Озеро Росохан

Прибуваємо до найбільшого озера Ґорґан. На озері є вільний час для фотографій та кавування. Насолоджуємося неймовірними краєвидами!

14:00

Фотосесія та відпочинок

Час для незабутніх фотографій на тлі озера та гір. Можливість насолодитися гарячою кавою та перекусом.

16:00

Повернення до машини

Після озера повертаємось до машини тією ж мальовничою стежкою через горганський ліс.

21:00

Повернення у Франківськ

Прибуття у Франківськ о 21:00.

23:00

Повернення в Чернівці

Прибуття в Чернівці о 23:00, сповнені незабутніх вражень від найбільшого озера Ґорґан.

Що взяти з собою

  • Зручне взуття для походу
  • Теплий одяг (в горах може бути прохолодно)
  • Фотоапарат для незабутніх кадрів
  • Термос з гарячим напоєм
  • Перекус для енергії
  • Сонцезахисний крем
